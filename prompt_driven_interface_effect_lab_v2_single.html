The Prompt‑Driven Interface

This single-file demo teaches webdesign with AI agents by making the UI demonstrate the effects it describes. Use the Agent Control Center to activate motion, depth, scroll engines, typography, cursor systems and accessibility guardrails.

01 / Prompting Levels

Lessons that change the page itself.

Zero-shot, few-shot and agentic workflow are not only explained. Each mode transforms the interface so users can feel the design consequence of the prompt style.

1.1

Zero‑Shot Nature Distilled

Activates a stripped-down earth-tone layer. Canvas noise and maximal motion recede, creating a clean, sober, content-first interface.

1.2

Few‑Shot Tactility

Buttons and cards become physically responsive: they pull toward the cursor and demonstrate a promptable magnetic interaction.

1.3

Agentic Bento FLIP

The bento grid rearranges density like an agent-generated layout revision while preserving spatial continuity.

05 / Accessibility

Motion needs guardrails.

Reduced motion is implemented as a real simulation: parallax, magnetic pull, 3D rotation and cursor trails flatten without removing the information architecture.

Accessible by design

The switch in the Agent Control Center simulates prefers-reduced-motion. Effects remain understandable, just not animated.
